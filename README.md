# Driftline

A lightweight Three.js arcade game designed for one-thumb play in a mobile browser.

## Play online

**https://teejay.dev/driftline-game/**

For the best mobile experience, open it in Safari or Chrome and rotate the phone however you prefer.

## Run locally

```bash
npm install
npm run dev
```

Open the URL Vite prints (normally `http://localhost:5173/`).

## Controls

- **Touch / mouse:** press and drag anywhere on the course to steer.
- **Keyboard:** Left/Right arrows or A/D.
- The run pauses when the tab becomes hidden.

## Checks

```bash
npm run check
```

The production build is emitted to `dist/` and can be served as a static site.
